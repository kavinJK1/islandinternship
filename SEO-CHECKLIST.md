# SEO Checklist

- [ ] Update canonical URL to final production domain.
- [ ] Update OG URL and OG image URL to production paths.
- [ ] Confirm title and meta description reflect final positioning.
- [ ] Validate schema JSON-LD (Organization, Service, FAQPage).
- [ ] Submit sitemap in Google Search Console (if added later).
- [ ] Verify no broken internal links.
- [ ] Add final favicon set (ICO + Apple + optional PNG sizes).
- [ ] Replace placeholder GA4 measurement ID.
- [ ] Add robots.txt if needed.
- [ ] Ensure privacy and terms pages are finalized.
