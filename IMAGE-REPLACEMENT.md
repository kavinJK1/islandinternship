# Image Replacement Guide

## Current placeholders
- Hero background: Unsplash remote URL in `index.html`
- Local placeholders in `images/`:
  - `hero-bali.jpg`
  - `map-bali.svg`
  - `map-srilanka.svg`
  - `mindmap-1.svg`
  - `mindmap-2.svg`
  - `mindmap-3.svg`

## Replace process
1. Keep the same filenames to avoid HTML edits.
2. Recommended dimensions:
   - `hero-bali.jpg`: 1920x1080 (WebP/JPG, <350 KB)
   - map/mindmaps: SVG preferred, or PNG <200 KB each
3. Compress images before upload.
4. Ensure every image has meaningful alt text in `index.html`.
