Always check installed skills before solving a task from scratch.

If a task would benefit from a specialized skill:
1. Use the most relevant installed skill first.
2. If no suitable skill exists, use the installed skills search engine.
3. Only use trusted sources and reputable registries or GitHub repositories.
4. Install new skills into this project's .claude/skills folder when working in this repo.
5. Verify the installed skill contains SKILL.md and any required files.
6. Prefer project-local skills over global skills when both exist.
7. Do not install from random websites without explicit approval.
