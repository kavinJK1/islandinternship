# Sell Like Crazy

This folder is a reusable conversion-focused skill pack for sharpening offers, messaging, pages, emails, funnels, and CTAs across different industries without tying the guidance to any single business.

The skill now follows a book-first workflow: `sell-like-crazy.pdf` is the primary source, and the markdown files support implementation, structure, and QA.

## Files

- `SKILL.md`: core entrypoint and operating rules for using the skill
- `sell-like-crazy.pdf`: primary source material from the book
- `framework.md`: diagnostic framework for message, offer, proof, and funnel decisions
- `conversion-patterns.md`: reusable copy and offer scaffolds
- `checklists.md`: quality-control checklists before shipping copy or strategy
- `anti-patterns.md`: mistakes that weaken conversion or trust
- `prompts.md`: prompt templates for audits, rewrites, and strategy work
- `context.md`: intake template for gathering the right inputs

## Suggested workflow

1. Start with `SKILL.md` for the operating rules and source priority.
2. Consult `sell-like-crazy.pdf` first for the relevant principle, framework, or tactic.
3. Use `context.md` to collect the minimum business context.
4. Pull from `framework.md`, `conversion-patterns.md`, and `prompts.md` to turn the book's ideas into practical output.
5. Run the result through `checklists.md` and `anti-patterns.md` before finalizing.

## Notes

- The content is intentionally business-agnostic and reusable.
- It is designed to produce practical output, not vague theory.
- The PDF remains intact and is not overwritten by this package.
- If Claude cannot access the PDF in practice, attaching the PDF in-chat is the most reliable fallback.
