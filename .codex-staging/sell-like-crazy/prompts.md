# Prompt Templates

Replace bracketed placeholders with real context. If context is thin, pair these with the intake template in `context.md`.

## 1. Conversion audit

```text
Act as a direct-response conversion strategist.

Audit this asset for clarity, relevance, proof, offer strength, friction, and CTA quality:
[paste page, email, ad, script, or URL notes]

Context:
- Audience: [audience]
- Goal: [click / opt-in / book / buy / reply]
- Offer: [offer]
- Price or ask: [price or CTA]
- Known objections: [objections]
- Available proof: [proof]

Return:
1. The top 3 conversion blockers in order
2. A rewritten version of the highest-leverage section
3. 3 test ideas ranked by likely impact
```
